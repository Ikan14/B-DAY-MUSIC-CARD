# B-DAY-MUSIC-CARD
A simply lovely B-Day Music Card With ARDUINO UNO

## Description 
 A simple Birthday Music box that showing your little running text with 'Happy Birthday' song to lighten ye mood! With arduino Uno

## Features :D
- Shows running text for your b-day person in the LCD
- Produce buzzing sound that singing 'Happy Birthday" song
- 3 Red LEDS to ligthen the mood

## Component I use
- Arduino UNO R3
- Piezo Buzzer
- Small breadboard
- Red LEDs
- 9V Battery
- Module LCD 16x2 I2C
- Resistor 330 Ohm
- Push button

## Firmware
Firmware is on the /Firmware folder and it is written in C++ with Arduino IDE framework
